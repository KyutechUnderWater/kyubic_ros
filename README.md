# kyubic_ros

## Requirement
- Docker >= 27.5.1
- Docker Compose >= 2.32.4

## Installation
Create docker container and Setup continer (user password, enviroment variable, alias, and udev rule)
~~~bash
. install.sh {your_password}
~~~

## How to Use
Start container and Run bash
~~~bash
ros2_start
~~~

## Uninstall
~~~bash
. uninstall.sh
~~~
